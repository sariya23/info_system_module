# Практическая работа №6

## Тема

Автоматизация тестирования для модуля из ПЗ 4-5.

## Выполненные задания

### Задание 1

Добавлены автотесты на Python с использованием `unittest`, проверяющие входные данные модуля:

- обработка числового ввода;
- обработка строкового ввода;
- обработка пустой строки;
- чтение данных из файла;
- обработка отсутствующего файла;
- обработка корректной ссылки;
- обработка пустой ссылки;
- выбор типа ввода из меню;
- завершение программы при неизвестном типе ввода.

Файл с тестами: `tests/test_main.py`.

### Задание 2

Настроен `CI/CD` через GitHub Actions.

Workflow расположен в файле `.github/workflows/python-tests.yml` и запускает автотесты:

- при push в ветки `main` и `master`;
- при pull request в ветки `main` и `master`.

### Задание 3

Работоспособность модуля проверена локальным запуском автотестов командой:

```bash
python3 -m unittest discover -s tests -v
```

### Задание 4

Результаты выполнения добавлены в данный отчет.

## Структура добавленных файлов

- `tests/test_main.py` - автотесты;
- `.github/workflows/python-tests.yml` - workflow для автоматического запуска тестов;
- `README.md` - отчет по практической работе.

## Результат

Автотесты написаны, CI настроен, проект подготовлен к проверке при отправке изменений в удаленный репозиторий.
